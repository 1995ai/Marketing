## How This Repo Works

- **Sentinel** writes research outputs and draft content here directly (overnight agents)
- **Matthew + Jeff** review, edit, and approve before publishing
- Staging-ready content is marked with `STATUS: Ready for review` at the top of the file
- Published content moves to an `archive/` subfolder
